NFL Analytics Project

By: Tyran Johnson:

Introduction:

This project delves into the analysis of NFL data spanning from 2018 to 2022, with the aim of uncovering insights into team and player performance, as well as game outcomes. By examining statistical metrics, the purpose was to seek to identify the key factors that drive success in the NFL.

In addition to revealing interesting facts and insights within football metrics, this project also serves as an innovative practice tool for inexperienced developers. It offers a hands-on opportunity to better understand some of the capabilities of data analysis using Python.
